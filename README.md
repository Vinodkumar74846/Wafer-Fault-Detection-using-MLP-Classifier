# 🧪 Wafer Fault Detection using Machine Learning

This project focuses on detecting **faults in semiconductor wafers** using Machine Learning techniques.  
The goal is to identify defective wafers early in the manufacturing process to improve product quality and reduce production costs.

---

## 📌 Project Overview

Wafer fault detection is an important task in the **semiconductor manufacturing industry**.  
Machine Learning models can analyze wafer data and identify patterns that indicate faults or defects.

This project demonstrates how data analysis and machine learning can be used to classify wafers as **faulty or non-faulty**.
